ListQuee - Web Todolist Sederhana Catatan Harian

Web berbasis PHP & MySQL untuk menambahkan catatan harian.

Tentang Proyek

Proyek ini dibuat dengan tujuan agar pengguna dapat menambahkan catatan harian dengan sistem reward point. Sehingga pengguna termotivasi untuk menambahkan catatan harian. Selain itu, proyek ini juga memperdalam pemahaman saya terkait PHP dan MySQL.

Fitur

  • Integrasi CRUD (Create, Read, Update, Delete) Catatan.
  • Sistem Login & Registrasi User.
  • Sistem Reward Point.
